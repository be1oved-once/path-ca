answer checking

Live Test System

Real examination
discipline, online.

PathCA live tests are not just timed - they are actively monitored. Every action that breaks exam discipline is detected and handled automatically.

Admin creates the live test

Custom questions (MCQ or written), custom duration, published instantly. Students see the test option immediately.

Active monitoring starts

The moment a student enters, the system tracks tab focus, app switching, visibility changes and split-screen behaviour.

Cheating triggers an instant penalty

Leaving the tab, opening another app, or backgrounding the test locks the attempt immediately with a recorded violation.

Admin sees everything

Live leaderboard showing ranks, marks, violations and detailed answers - all in one place, in real time.

No refresh tricks. No multiple tabs. The test behaves like a real examination hall.
